[![Python](https://img.shields.io/badge/Python-3.10%2B-blue.svg)](https://www.python.org/)

[![GitHub](https://img.shields.io/badge/GitHub-OpenGHz%2Fauto--atomic--operation-181717?logo=github)](https://github.com/OpenGHz/auto-atomic-operation)

- Declarative YAML task definitions
- Hydra-powered configuration
- Pluggable backends (MuJoCo, real robot, mock)
- Pose randomization with collision avoidance
- 3D Gaussian Splatting rendering
